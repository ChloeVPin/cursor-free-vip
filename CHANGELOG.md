# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-25

### Added
- Initial release of Cursor Free VIP
- Modern GUI built with Electron & Vue.js
- Reset Machine ID functionality
- Multi-Account Manager (Beta) - Create, import, switch, and delete accounts
- Manual Authentication feature
- Disable Auto-Updates option
- Token Limit Helper
- Quit Cursor Process functionality
- Totally Reset Cursor feature
- Built-in Documentation page
- In-App Fixes for common Cursor issues:
  - Fix missing `workbench.desktop.main.js` file
  - Fix missing `product.json` file
  - Fix Cursor installation location issues
- Multi-language support (English, 简体中文, Русский)
- Modern dark theme with glassmorphism interface
- Real-time progress tracking with detailed logs
- Portable Windows executable (no installation required)

### Notes
- Windows (x64) officially released and tested
- macOS and Linux support coming soon (build from source available)
- Multi-Account Manager is in beta and untested by developer
- Educational purposes only
